[[Beyond the Veil/Veil Glossary/Auditory\|Auditory]] actions and effects rely on sound. An action with the auditory trait can be successfully performed only if the creature using the action can speak or otherwise produce the required sounds. An ability or effect with the auditory trait has its effect only if the target can hear it. This applies only to sound-based parts of the effect, as determined by the [[Beyond the Veil/Veil Glossary/Director\|Director]]. [[Beyond the Veil/Veil Glossary/Auditory\|Auditory]] effects ignore all cover.
