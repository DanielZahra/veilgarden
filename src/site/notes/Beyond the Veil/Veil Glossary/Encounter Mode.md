[[Beyond the Veil/Veil Glossary/Encounter Mode\|Encounter Mode]] is used for when every round, turn and decision needs to be measured as each action has impact. This mode is used mainly for combat, complex traps and or some of the skill challenges which require action by action use.

[[Beyond the Veil/Veil Glossary/Encounter Mode\|Encounter Mode]] is measured in rounds and turns. A turn is when a single creature uses its turn. When all creatures have taken their turn, a round is completed. The order in which each creature acts is handled by [[Beyond the Veil/Veil Glossary/Initiative\|Initiative]].
