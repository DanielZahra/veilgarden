### [[Beyond the Veil/Veil Glossary/Group Skill Check\|Group Skill Check]] 

When an event calls for a [[Beyond the Veil/Veil Glossary/Group Skill Check\|Group Skill Check]], you (the party) all roll against 1 or more DCs, and each success grants ‘points’ towards a success. Each failure grants a failure point. At the end of the skill challenge, if the number of successes is equal or higher than failure, you succeed.

| **Degree**       | **Effect**           |
| ---------------- | -------------------- |
| Critical Success | You gain 2 successes |
| Success          | You gain 1 success   |
| Failure          | You gain a failure   |
| Critical Failure | You gain 2 failures  |
