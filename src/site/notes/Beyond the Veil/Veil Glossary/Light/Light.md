When an objects emits light, vision can be used. Anything within the [[Beyond the Veil/Veil Glossary/Light/Light\|Light]] can be seen normally.
{ #d374e8}

Sources of [[Beyond the Veil/Veil Glossary/Light/Light\|Light]] emit additional range in [[Beyond the Veil/Veil Glossary/Dim Light\|Dim Light]] (e.g. a source of 6 [[Beyond the Veil/Veil Glossary/Light/Light\|Light]] extends to an additional 6 [[Beyond the Veil/Veil Glossary/Dim Light\|Dim Light]])
{ #566173}

[[Beyond the Veil/Veil Glossary/Light/Light\|Light]] is followed by a number indicating how far the lights reaches. You can double this number to also see how far the [[Beyond the Veil/Veil Glossary/Dim Light\|Dim Light]] reaches.
{ #a3ce5d}

[[Beyond the Veil/Veil Glossary/Light/Light\|Light]] source typically functions like an [[Beyond the Veil/Veil Glossary/Emanation/Emanation\|Emanation]] unless indicated otherwise.
{ #c0c95a}
