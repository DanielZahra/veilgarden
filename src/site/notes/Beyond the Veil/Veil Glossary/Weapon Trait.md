[[Beyond the Veil/Veil Glossary/Weapon Trait\|Weapon Trait]] are rules applied to weapon usage. These traits define how much damage a weapon does, how it is wielded and any effect applied when the weapon is used. Weapons cannot have the same trait type applied multiple times, instead if the same trait is applied multiple times the strongest version is used.
